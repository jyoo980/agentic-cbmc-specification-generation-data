# Data Package for "What Can Agents Prove About C Programs?"

This repository contains the subject programs used to evaluate an agentic
  approach to CBMC specification generation and verification.

## Index

- [minus-harness](./minus-harness): Contains subject programs
  with specifiations without the harness.
- [minus-mutation-testing](./minus-mutation-testing): Contains subject programs
  with specifiations without mutation testing.
- [minus-docs](./minus-docs): Contains subject programs
  with specifiations generated without curated documentation.
- [minus-tools](./minus-tools): Contains subject programs
  with specifiations generated without tools.
- [pure-agentic-baseline](./pure-agentic-baseline): Contains subject programs
  with specifiations generated by Claude Code (the "pure agentic" baseline).
- [full](./full): Contains subject programs
  with specifiations with all components in-place.

## Scripts

- [scripts/generate_mutation_kill_scores.py](./scripts/generate_mutation_kill_scores.py): Generates the rows of Table IV.
- [scripts/generate_verification_counts_table.py](./scripts/generate_verification_counts_table.py): Generates the rows of Table III.
